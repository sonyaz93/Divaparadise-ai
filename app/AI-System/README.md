## üß† System Core & Administration

Beyond the functional modules, the system includes centralized management layers.

### 1. [`Generative Language`](./Generative%20Language) (The "Admin Layer")
The system administration interface for the Gemini API.
*   **`api/model_registry.py`**: Lists available models and their limits.
*   **`api/cache_manager.py`**: Manages cached content to save tokens.
*   **`api/batch_manager.py`**: Handles large-scale batch processing jobs.

### 2. [`The Models`](./The%20Models) (The "Knowledge Base")
The single source of truth for system intelligence.
*   **`model_manifest.json`**: A registry of every active model (Cloud & Local).
*   **`system_prompts/`**: Central repository for System Instructions (Personas).
*   **`weights/`**: Storage for local model binaries (Wav2Lip, etc.).

### 3. [`99_Multimodal_Lab`](./99_Multimodal_Lab) (The "Playground")
A testing ground for combining multiple modalities (e.g., Image + Audio + Thinking) into complex workflows.
